🔧 Common Issues & Solutions

Issue: "Failed to fetch" or CORS error

Solution: Edge Functions are not deployed or URL is wrong

cd supabase
npx supabase login
./deploy.sh

Issue: Data saved but no email

Solution: RESEND_API_KEY not set or invalid

  1. Check email_log table for error messages
  2. Verify RESEND_API_KEY in Edge Function secrets
  3. Test API key at Resend Dashboard

Issue: No data saved, no email

Solution: Frontend not calling Edge Functions

  1. Open browser DevTools Console
  2. Submit form and watch for errors
  3. Check Network tab for API calls

Issue: 401/403 errors

Solution: Auth token issues

  1. Check VITE_SUPABASE_URL and VITE_SUPABASE_PUBLISHABLE_DEFAULT_KEY in .env.local
  2. Restart dev server after changing .env
